# Plataforma ONG Exemplo — Atividade 2 (CSS3)

Segunda entrega do projeto de Front-End, focada na **organização e aplicação de CSS3**.  
O objetivo é transformar a estrutura HTML5 anterior em uma interface visual profissional, responsiva e acessível.

---

## Objetivo
Aplicar os conceitos avançados de CSS3:
- Sistema de design com variáveis customizadas;
- Tipografia e espaçamento hierárquicos;
- Layout responsivo com **Flexbox** e **Grid**;
- Navegação mobile com **menu hambúrguer interativo**;
- Componentes reutilizáveis (cards, botões, formulários).
